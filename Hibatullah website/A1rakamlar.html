Queue (Kuyruk) Soruları :

a) İlk giren ilk çıkar (FIFO) prensibine göre çalışan bir veri yapısıdır.
b) Son giren ilk çıkar (LIFO) prensibine göre çalışan bir veri yapısıdır.
c) Rastgele erişim ile çalışan bir veri yapısıdır.
class Queue {
    int front, rear, size;
    int* array;
public:
    Queue(int s) {
        front = rear = -1;
        size = s;
        array = new int[s];
    }
    void enqueue(int value) {
        if (rear == size - 1) {
            cout << "Queue is full";
            return;
        }
        if (front == -1) front = 0;
        array[++rear] = value;
    }
};
a) Evet
b) Hayır, rear kontrolü yanlış
c) Hayır, front kontrolü yanlış
a) İlk giren ilk çıkar (FIFO) prensibine göre çalışan ve kuyruğun sonundan tekrar başa dönen bir veri yapısıdır.
b) Son giren ilk çıkar (LIFO) prensibine göre çalışan bir veri yapısıdır.
c) Rastgele erişim ile çalışan bir veri yapısıdır.
class CircularQueue {
    int front, rear, size;
    int* array;
public:
    CircularQueue(int s) {
        front = rear = -1;
        size = s;
        array = new int[s];
    }
    void enqueue(int value) {
        if ((rear + 1) % size == front) {
            cout << "Queue is full";
            return;
        }
        if (front == -1) front = 0;
        rear = (rear + 1) % size;
        array[rear] = value;
    }
};
a) Evet
b) Hayır, rear kontrolü yanlış
c) Hayır, front kontrolü yanlış
